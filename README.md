# Device Preview (com.actionfit.devicepreview)

여러 디바이스 해상도 기준으로 현재 UI를 미리보는 Unity 에디터 윈도우입니다.

## 설치 (manifest.json, Git URL)

```json
{
  "dependencies": {
    "com.actionfit.devicepreview": "https://github.com/ActionFit-Editor/Device_Preview.git#1.0.1"
  }
}
```

## 구성

- **Editor** (`com.actionfit.devicepreview.Editor`): `DevicePreviewWindow`.

## 사용

`Tools > JH Tools > Each Device Check UI Preview` (단축키 **Shift+Alt+P**)로 윈도우를 엽니다.
